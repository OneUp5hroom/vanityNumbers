About this site

This site was created as a technical exersise.
Please feel free to play around with the site and generate some vanity numbers for your own number!

Note:
When using the number generator, if there is already a record for the entered phone number ther response will be quick and return an existing database entry. If it has to generate a new number it will take a few seconds, and I have not been able to add a loading wheel / bar so far!
